# ML Microservice for Face Attendance

This service handles Face Detection, Recognition, and Anti-Spoofing.

## Hosting on Hugging Face Spaces (Docker)
1. Create a new Space.
2. Select **Docker** as the SDK.
3. Choose the **Blank** template.
4. Upload all files from the `ml_service` folder.
5. Hugging Face will use the `Dockerfile` automatically.
